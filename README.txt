Reviewder - the review downloader

This is version 0.1. You can only download complete review bundles (all reviews by and on a single DCI number).
Review viewing privileges in the Judge Center are required. You should have those if you're an RC or L4+.

INSTALLATION
- unzip reviewder_win.zip into c:\reviewder

USAGE
- execute c:\reviewder\main
- follow instructions
- results are saved in <dci_number>_reviews.html in your Documents folder
